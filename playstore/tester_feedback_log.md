# Tiny Next Closed Test Feedback Log

Use this file during the 12-tester, 14-day closed test. Keep entries concise and factual because Play Console asks about the testing process and production readiness when personal accounts apply for production access.

## Tester Instructions

Ask each tester to:

1. Join the closed test with the Google account added in Play Console.
2. Install Tiny Next from the Play testing link.
3. Complete onboarding.
4. Add at least one custom task.
5. Use "Pick My Next Task."
6. Try Done, Skip, Snooze, and Another.
7. Open Stats, Task List, Premium, and Settings.
8. Report one thing that worked, one confusing point, and the device model.
